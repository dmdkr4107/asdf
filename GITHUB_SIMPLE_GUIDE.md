
---

## ⚡ 초간단 방법 (3분 소요)

### **단계 1️⃣: GitHub 열기**

브라우저에서 이 링크 열기:
```
https://github.com/wshobson/agents
```

**또는** 내 저장소가 있다면 그 저장소 링크

---

### **단계 2️⃣: agents-main 폴더 들어가기**

1. 저장소 페이지에서 "agents-main" 폴더 찾기
2. 클릭해서 폴더 안으로 이동
3. URL이 이렇게 변함:
   ```
   https://github.com/wshobson/agents/tree/main/agents-main
   ```

---

### **단계 3️⃣: "Add file" 클릭**

1. 폴더 안에서 **"Add file"** 버튼 찾기 (오른쪽 위)
2. 드롭다운 메뉴 나타남
3. **"Upload files"** 클릭

```
┌─────────────────┐
│  Add file ▼     │ ← 클릭!
├─────────────────┤
│ Create new file │
│ Upload files    │ ← 이것 클릭!
│ Copy to new file│
└─────────────────┘
```

---

### **단계 4️⃣: 파일 선택하기**

**방법 A: 드래그-앤-드롭 (가장 쉬움!)**
```
1. Windows 탐색기 열기
2. 여기로 이동:
   C:\Users\user\dmdkr\agents-main\agents-main\

3. 이 3개 파일을 보면:
   ✓ ORCHESTRATION.md
   ✓ orchestrate.py
   ✓ QUICK_REFERENCE.md

4. 이 파일들을 GitHub 웹 페이지로 드래그!
   (마우스로 끌어서 떨어뜨림)
```

**방법 B: 파일 선택 대화**
```
1. "Upload files" 클릭
2. "파일 선택" 버튼 나타남
3. 3개 파일을 선택
   (Ctrl+Click으로 여러 개 선택)
4. "열기" 클릭
```

## 🆘 문제 해결

**Q: "Upload files" 버튼이 없어요**
```
→ GitHub 로그인했는지 확인
→ 저장소에 쓰기 권한이 있는지 확인
→ Fork한 저장소면 fork에 올려야 함
```

**Q: 파일을 찾을 수 없어요**
```
→ 경로 확인: C:\Users\user\dmdkr\agents-main\agents-main\
→ 파일명 정확히 확인:
   ✓ ORCHESTRATION.md (대문자)
   ✓ orchestrate.py (소문자)
   ✓ QUICK_REFERENCE.md (대문자)
```

**Q: 업로드 중 에러가 났어요**
```
→ 인터넷 연결 확인
→ 파일 크기 확인 (모두 20KB 미만 - OK)
→ 다시 시도
```
